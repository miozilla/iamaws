## Objectives
- To explore pre-created IAM Users & Groups
- To inspect IAM policies as applied to pre-created groups
- To perform real-world scenario, adding users to groups with specific capabilities enabled
- To locate and using the IAM sign-in URL
- To experiment with the effects of policies on service access
